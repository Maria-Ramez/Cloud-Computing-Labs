Exploring Cloud Virtualization and Data Center Architecture

Part A: VM vs Container
In this lab, a Docker container was used to analyze system resource usage.

Commands used:
free -h
ps aux --sort=-%mem | head
df -h

Observation:
Containers start quickly and use fewer resources.
They run fewer processes compared to virtual machines.

Conclusion:
Containers are lightweight and efficient, while virtual machines provide stronger isolation but consume more resources.
